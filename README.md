# SOC Detection and Monitoring Lab
A hands-on SOC Detection and Monitoring Lab using Splunk Enterprise, Sysmon, Windows 10, Kali Linux, and Ubuntu Server.

## Project Overview

This repository documents the design, implementation, and operation of a Security Operations Center (SOC) Detection and Monitoring Lab. The objective of this project is to gain practical experience with security monitoring, log collection, attack detection, threat investigation, and incident response using industry-standard tools.

The lab is being built step by step and documented throughout the entire process.

---

## Project Objectives

- Build a complete SOC home lab.
- Configure a centralized SIEM using Splunk Enterprise.
- Collect Windows Event Logs and Sysmon logs.
- Simulate attacks from a Kali Linux machine.
- Create SPL searches and detection rules.
- Investigate security events.
- Build dashboards and alerts.
- Document every stage of the project.
